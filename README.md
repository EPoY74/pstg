# Pump Station Telemetry Gateway (Industrial)

Pump Station Telemetry Gateway — сервис/демон для промышленной телеметрии насосных станций (так же можно использовать для мониторинга насосных станций водоканала и КНС) в OT/SCADA-контуре: собирает значения и состояния через подключаемые драйверы (например, Modbus RTU/TCP через RTU↔Ethernet шлюзы или эмулятор), приводит сигналы к каноническому формату и сохраняет их в SQL-хранилище как временные ряды. Проект учитывает реальность эксплуатации (разрывы связи, частичные данные, неравномерный опрос), поэтому архитектурно заложены плагинные драйверы, симулятор для разработки и контрактные тесты на совместимость формата данных.

Сервис предоставляет REST API для интеграций и дашбордов: выборки time-series и базовые тренды, события (events), аварии/алармы (alarms) и аудит действий (audit log). UI можно вынести в отдельный веб-дашборд или использовать внешние инструменты наблюдаемости (например Grafana), а CLI применяется как админ-инструмент для миграций и обслуживания (migrate/seed/health/simulate/import/export).

Разработка ведётся итеративно: MVP “сбор → хранение → API → тренды”, затем правила алармов, журналирование, роли и права доступа, команды управления и HA-сценарии; каждую неделю — небольшой вертикальный релиз с миграциями и тестами.
